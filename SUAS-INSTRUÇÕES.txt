1. Realizar instalação do Ruby e Ruby on Rails. Não é necessário instalar nenhums dos BDs (https://gorails.com/setup)

2. Navegar até o diretório do projeto e executar o seguinte comando para executar o sistema:
    rails db:create && rails db:migrate && rails s

3. Apos subir o servidor, acessar a url localhost:3000

5. Para rodar os testes, executar o seguinte comando:
    bundle exec rspec

5. Qualquer dúvida a disposição.
